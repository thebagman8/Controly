## 🎯 What is Controly?

**Controly** turns your phone into a wireless remote control for your computer — no app installation required. Just open a webpage from your phone's browser, and you're set.

Perfect for a PC connected to your TV in the living room — control it from the couch without getting up for the keyboard.

## ❓ FAQ

<details>
<summary><b>Do I need to install anything on my phone?</b></summary>
<br/>
No — Controly works entirely through your phone's web browser.
</details>

<details>
<summary><b>Why does Windows show a warning when I run it?</b></summary>
<br/>
The app isn't digitally signed yet, so Windows SmartScreen flags it by default. This doesn't mean it's unsafe — just that it hasn't gone through Microsoft's paid signing process. Click "More info" → "Run anyway" to continue.
</details>

<details>
<summary><b>Does my phone need to be on the same network as my PC?</b></summary>
<br/>
Yes, both devices need to be connected to the same Wi-Fi network for Controly to work.
</details>
